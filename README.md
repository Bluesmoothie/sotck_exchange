# stock_exchange

Display stock exchange using [Finnhub](https://finnhub.io/) API

## Dependencies

## Dev - dependencies

```bash
make g++ sdl2-dev libjsoncpp-dev libcurl4-openssl-dev libsecret-1-dev libglib2.0-dev pkg-config
```

## Runtime dependencies

```bash
libsecret-1-0
```

## Used libraries

- [ImGui](https://github.com/ocornut/imgui)
- [finnhub-api-cpp](https://github.com/calmdaysamuel/finnhub-api-cpp)
